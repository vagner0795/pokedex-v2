### Sobre

O projeto se trata de uma pokedex com pokemons da 1° a 5° geração. também possui input de pesquisa e botões para avançar e voltar pokemons. usando uma API, foi usado:

- [x] Responsividade
- [x] JAVASCRIPT
- [x] API
